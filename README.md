# CRUD Todo App (Node.js + Express)

## Description
This is a simple CRUD Todo app built with Node.js and Express.  
It uses an **array** to store data instead of a database (as per assignment).

## Features
- Create a todo
- Read all todos
- Update a todo by ID
- Delete a todo by ID
